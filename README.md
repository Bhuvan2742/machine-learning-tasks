# Machine Learning Tasks

This repository contains simple machine learning practice notebooks implemented using Python. Each notebook focuses on different fundamental machine learning concepts, from data preprocessing to model evaluation.

## Files

- **ML1.ipynb**  
  Covers basic machine learning tasks such as data loading, preprocessing, model training, and evaluation.

- **ML Assignment 1.ipynb**  
  Focuses on basic data handling, preprocessing, and exploratory data analysis.

- **ML Assignment 2.ipynb**  
  Includes feature selection, data transformation, and applying basic machine learning algorithms.

- **ML Assignment 3.ipynb**  
  Demonstrates model training, train-test split, and performance evaluation.

- **ML Assignment 4.ipynb**  
  Focuses on model comparison, performance metrics, and result analysis.

## Tools Used
- Python  
- NumPy  
- Pandas  
- Scikit-learn  
- Matplotlib  

## Purpose
To practice and understand fundamental machine learning concepts and gain hands-on experience with end-to-end machine learning workflows.
